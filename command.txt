git init   ----(Create git project)
git add README.md  (Add untrack to stage area)
git status (check status)
git remote add origin repoName (Link out local project to Github)
git push origin master( Local files availbe on Github)
git clone repoName
git pull origin master(Take code from repository to local if any new file added  in repository)
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/SenchaArchitect12345/Github-learning.git
git push -u origin main



Open the command line.

Set your username:
git config --global user.name "Ashish"

Set your email address:
git config --global user.email "abc@gmail.com"
